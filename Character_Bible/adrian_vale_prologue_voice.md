## What Musk-class sounds like **in prose** (not dialogue transcript)

### Do

| Pattern | Example register |
|---------|------------------|
| **First principles** | *If you strip it down, you're not sending a person. You're correlating state.* |
| **Blunt short sentences** | *You correlated it. Full stop.* |
| **Self-interrupt** | *The module landed—look, fourteen months ago, which is insane, but—* |
| **Dark joke that dies** | *He'd made a bankruptcy joke. The room died. Fair.* |
| **Technical noun stacks** | *entangled pair lattice, implant crown, quantum repeater trunk* |
| **Grand + shabby** | *cathedral / personality disorder / selling futures on futures* |
| **SA-adjacent vocabulary (sparse)** | *rubbish, quite, maths, bloody (rare), straight away, sort out* |
| **Frantic listing** | *launch early, launch again, satellites, boards, collapse* |
| **Rhetorical questions to self** | *So what does that mean? It means geography stops mattering for attention.* |
| **Understatement → explosion** | *Not ideal. / Historic.* |

### Don't

| Avoid | Why |
|-------|-----|
| Phonetic accent (*"yah no"*) | Caricature |
| Literary prestige metaphor every paragraph | Sounds like **Mara**, not Vale |
| US tech-bro MBA smooth | Wrong class of geek |
| Perfect TED talk prose | He's exhausted, messy, **live** |
| Naming real Musk companies/people | In-world only |
| DMT / drugs in prologue | Later arc |

## SA → English markers (use **1–2 per page max**)

- **Directness:** less hedging (*"The critics were wrong"* not *"The critics had perhaps misjudged"*)
- **British lean:** favour, metres (when counting), maths (interior only)
- **Rhythm:** occasional **comma splice** or **dash stack** when manic (stress → 7)
- **Emphasis:** *quite*, *actually*, *basically*, *look*
- **Not British-only:** keep US setting words (Nevada, elevator ok)
