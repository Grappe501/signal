# THE SIGNAL CYCLE

# BOOK TWO · *THE GREAT DISCONNECTION*

# MOVEMENT ARCHITECTURE

**Thesis:** **CONTROL**

**Total movements:** **22** (3–4 per act)

**Status:** Set piece architecture complete · Chapters/scenes **not started**

**Mission:** Break six-act skeleton into **movements** — larger than chapter, smaller than act. Every movement **pulls toward or away from** a signature set piece.

**Date:** 2026-05-30

---

> Book Two is not one collapse. It is a society agreeing, step by step, to become **less free** because freedom requires trust — and trust has become **too expensive**.
