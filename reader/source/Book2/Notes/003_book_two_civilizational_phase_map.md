# PHASE 3 · CLASSIFICATION

## Civilization experience

Uncertainty becomes intolerable. Everything receives a **label**.

## Label grammar (examples)

- Identity: **verified / pending / anomalous / unresolved**  
- Second: **active / idle / flagged / mirrored / offline-premium**  
- Risk: **HVI sub-bands · volatility cluster · reliability score**  
- Travel: **green / amber / manual-only / null zone**  
- Reliability: **within tolerance / strained / deprecated channel**  
- Upload exposure: **clean / exposed / carrier / unknown seed**  
- Faith/off-feed: **registered / unmapped / volatility**  

## Phase question answered

**When does classification become control?**

**The moment labels route resources faster than persons can appeal.**

Triggers:
- Amber travel → de facto ban without ban language  
- **Unresolved** idle → legal non-personhood workflow  
- **Carrier** upload label → employment/HVI cascade  
- **Unmapped** faith → excluded from disaster routing  
- **Offline-premium** sever → priced exit only elites afford  

Classification **stops being description** when it **precedes** experience — citizen arrives already sorted.

## Self-sort grammar (examples)

- **Mapped faith** vs **Off-map gathering**
