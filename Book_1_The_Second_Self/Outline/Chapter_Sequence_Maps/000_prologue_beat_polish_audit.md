# Prologue — Beat Polish Audit (Visual Programming Pass)

**Purpose:** Before **LOCK**, each beat must program the same picture in reader, listener, and eventual camera.  
**Character lens:** **Adrian Vale = Elon-class archetype amplified** — more years, more capital, more monitored biometrics, same odd genius (awkward humor, extinction obsession, showmanship, pocket superstitions, sleep debt). **Not impersonation** — original myth in-world.  
**Enneagram (Elon map → Vale):** **5w6** — Investigator/Loyalist · stress **→7** (manic scatter) · growth **→8** (assertive savior trap). DMT/drug arc **withheld** here; prologue shows only **clinical stabilizers** (canon Seg 021).

**Decisions locked this pass:**
| # | Choice |
|---|--------|
| **Aftershock** | **Pure silence** after cold line — no school/trading floor/tablet |
| **Totem** | **Pocket rock** (dossier: collects rocks) — cut smuggled-vial cute beat; Elon = superstitious engineer talisman |
| **Bridge** | ***Twelve years later*** alone after **cut to black** — no Part I title page (preserves cold-open prestige; Ch 1 header carries) |
