# CORE PRINCIPLE (Locked)

# Enneagram type is a **conscious-level** personality architecture — not HVI, not Relay skill, not memory content.

Humans **move** under pressure and integration.  
Seconds **hold** core presentation.

This is the **only** reliable psychological difference between a Second and a total copy of its human — subtle on the surface, unmistakable in the body over time.

---

# WHAT ENNEAGRAM IS *IN-WORLD*

| Layer | Rule |
|-------|------|
| **Not scored** | No official Enneagram number in LATTICE or HVI |
| **Not transmitted** | Relay syncs awareness states — not personality *movement* |
| **Felt, not labeled** | Characters never say "Type Four" on-page |
| **Craft tool** | Author/reader subconscious: *that person would act that way under stress* |

Enneagram describes **motivation**, **fear**, **default coping**, and **direction of change** when a human is safe vs threatened vs growing.

# HUMAN RULE (Locked)

Each human has:

| Field | Use in prose |
|-------|----------------|
| **Core type (1–9)** | Default desire / fear |
| **Wing** | Flavor — how the type *looks* in calm company |
| **Stress arrow** | Disintegration — who they **become briefly** under sustained threat, shame, overload |
| **Growth arrow** | Integration — who they **become briefly** when secure, honest, loved |

## On-page behavior

- **Calm scenes:** core + wing.
- **Stress scenes:** core **plus** stress-type tells (not a full costume change — a *leak*).
- **Growth beats:** rare, earned — integration arrow at climax or intimacy.
- **Never announce** the number. Let movement feel like empathy.

## Example (Mara 4w5)

| State | Reader feel |
|-------|-------------|
| **Core** | Observant, precise, allergic to performance, collects real things |
| **Stress → 2** | Reaches for usefulness / approval, then recoils — "don't comfort me" |
| **Growth → 1** | Principled truth without ornament — Ch 1 final line |

# SECOND RULE (Locked)

# A Second inherits the human's **core type and wing assignment** — but operates only in **Core Calm Presentation**.

| Second HAS | Second LACKS |
|------------|--------------|
| Core motivation structure | Stress-arrow **crossing** |
| Wing flavor at social best | Growth-arrow **integration** |
| Default reactions of the type | Shame spiral, numbing, sudden detachment, sudden aggression **that belong to the stress/growth types** |
| Consistent "best-day self" composure | The **psychological weather** humans hide poorly |

## Plain language

- A human **3** under scandal might go **9** — go flat, disappear, stop performing.
- **Crane-Second (3w2)** under the same scandal stays **3** — arranged patience, polished attention, *idle on the readout*.
- The copy is **not wrong**. It is **incomplete at the conscious level**.

## Idealization bias (Locked)

Seconds often read as **core type at social best**:

- less shame (4),
- less anxiety (6),
- less anger leak (8),
- less scattered escape (7).

Not a different type — **stress/growth noise filtered out**.  
This is why Mara's Second feels *more confident* than Mara. Why building liaison Seconds feel *too* courteous. Why Crane-Second looks like a **better** Julian.

# EXCEPTIONS (Locked)

## Vale-Second / THE EMPTY MAN

Total Continuity Transfer **removes the biological anchor** that enables conscious modulation.

| Layer | Result |
|-------|--------|
| Memory / IQ / type pattern | Present |
| Core calm presentation | **Flattens** |
| Stress / growth | **Absent** |
| Reader read | Wrong enneagram — **too directed**, no rest, no hesitate |

Not "Second rules" — **broken human consciousness wearing a type mask**.

## Rogue / protocol Seconds (Mercer-One, Mercer-Two)

Inherit Eli's **7w8** core presentation **plus** buried continuity protocol.  
Violence reads as **calm 7 strategic play**, not **stress→1** panic or **growth→5** wisdom.

## Hospitality / building Seconds

Generic shells — default **9w1** core calm (empty peacemaker welcome). Not full humans; no arc.

# CRAFT RULES FOR WRITERS

1. **Assign type before drafting dialogue** — especially under stress.
2. **Seconds:** ask *"What would this type do on their best day?"* — never *"What would they do if terrified?"*
3. **Contrast pairs on-page:** Mara (human 4 moving) beside Crane-Second (3 locked) beside Chen (1 leaking to 4 once).
4. **Do not exposition** Enneagram. No lectures. No in-world typology apps.
5. **Re-read stress scenes** — if everyone sounds stable, you've written Seconds, not humans.
